# AWS Test EC2 Instance Deployment (Amazon Linux)

## Overview

Deployed a virtual server on AWS using EC2 with Amazon Linux 2023.

## What I Did

* Selected and launched an Amazon Machine Image (AMI)
* Configured instance type (t3.nano) for cost efficiency
* Configured storage using gp3 volume (8 GB)
* Successfully launched and verified instance state
* Connected to the instance using EC2 Instance Connect

## Key Concepts Demonstrated

* EC2 instance provisioning
* AMI selection
* Networking and security groups
* Cloud-based Linux server access
